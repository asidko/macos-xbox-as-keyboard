# Changelog

## v2.0.0 (2026-03-27)

- Multiple profiles with color-coded indicators
- Macro system (key combos + text typing)
- Modifier key mappings (Cmd, Shift, Opt, Ctrl)
- Key dropdown with categorized sections
- Profile switching via controller button (configurable)
- Scrollable settings for large configurations
- Programmatic app icon

## v1.1.0 (2026-03-27)

- Settings dialog with configurable button-to-key mappings
- All 12 buttons mappable (D-pad, A/B/X/Y, LB/RB, LT/RT)
- Config persistence at `~/.config/xboxaskeyboard/config.json`

## v1.0.0 (2026-03-27)

- Initial release
- Menu bar app with on/off toggle
- D-pad to arrow keys mapping
- Controller auto-detection via GameController framework
- Background event monitoring + App Nap prevention
- Dev certificate for persistent Accessibility permission
